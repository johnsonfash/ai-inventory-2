# iOS implementation — PalliFcmPlugin.swift

This file is a placeholder. To finish the iOS push side:

## 1. Add Firebase to Apple Developer + Google Cloud

- In Firebase console, create an iOS app with bundle id `app.pallio`.
- Download `GoogleService-Info.plist` and drop it into
  `src-tauri/gen/apple/Pallio/` (after running `npm run tauri:ios:init`).
- In Apple Developer, enable "Push Notifications" capability for
  the App ID + create an APNs Auth Key (`.p8` file). Upload the key
  to Firebase project settings → Cloud Messaging.

## 2. Write the Swift plugin

Inside the Xcode project generated by `tauri ios init`, add a new
file `PalliFcmPlugin.swift` (skeleton below). Tauri 2 mobile
discovers Swift plugins under the project's source tree.

```swift
import Tauri
import Firebase
import FirebaseMessaging
import UserNotifications

@objc(PalliFcmPlugin)
public class PalliFcmPlugin: Plugin, MessagingDelegate, UNUserNotificationCenterDelegate {

  override public func load(webview: WKWebView) {
    FirebaseApp.configure()
    Messaging.messaging().delegate = self
    UNUserNotificationCenter.current().delegate = self
  }

  @objc public func registerForPush(_ invoke: Invoke) {
    UNUserNotificationCenter.current().requestAuthorization(
      options: [.alert, .badge, .sound]
    ) { granted, err in
      DispatchQueue.main.async {
        if granted {
          UIApplication.shared.registerForRemoteNotifications()
          Messaging.messaging().token { token, err in
            if let token = token {
              invoke.resolve(["token": token, "platform": "ios"])
            } else {
              invoke.reject(err?.localizedDescription ?? "no token")
            }
          }
        } else {
          invoke.reject("permission denied")
        }
      }
    }
  }

  // Forward incoming FCM messages to the JS side
  public func messaging(_ messaging: Messaging,
                         didReceiveRegistrationToken fcmToken: String?) {
    if let token = fcmToken {
      trigger("token-refresh", data: ["token": token])
    }
  }

  public func userNotificationCenter(_ center: UNUserNotificationCenter,
                                     willPresent notification: UNNotification,
                                     withCompletionHandler completionHandler:
                                       @escaping (UNNotificationPresentationOptions) -> Void) {
    trigger("push-message", data: notification.request.content.userInfo)
    completionHandler([.banner, .sound, .badge])
  }
}
```

## 3. Update Info.plist (in Xcode)

Add the `aps-environment` key with value `production` (or `development`
for dev signing) and the background modes:

```xml
<key>UIBackgroundModes</key>
<array>
  <string>remote-notification</string>
</array>
```

## 4. Wire from JS

```ts
import { invoke } from "@tauri-apps/api/core"

const { token } = await invoke<{ token: string; platform: string }>("plugin:pallio-fcm|register_for_push")
```
